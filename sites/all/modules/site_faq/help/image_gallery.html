Image Galleries are holders for groups of similar images to better assist with keeping your site's images organized.

Adding an Image Gallery

  • On the left side menu select Create Content - Image Galleries
  • Click on the Add Gallery tab
  • Enter a Name for the gallery
  • Enter a Description for the gallery. This will show up on the Image Gallery Listing page.
  • Pull down the Parent if you want the image gallery to be a Sub Gallery of another gallery.
  • Pull down the Weight. This determines the order on which this gallery will be shown on the Gallery Listing Page.
  • Send an email to webguy@domainnamereplacement.com that you have added a gallery to have the system display it properly in any Flash Slideshow viewers you may be using.

Once an image gallery has been added it will now show up in the Image Galleries pulldown on the Add/Edit an Image page
