To Add a Group/Committee

  • From the side menu select: Create Content - Committees
  • Enter a Name for the Group
  • Enter a short description/synopsis for the group. This appears on the page that lists all the groups/committees for the site.
  • Click on Menu Settings to show options:
    • Enter a link title
    • Under Parent Item: Select Committees (Under <Primary Menu> ) This will create a link on the top menu. It will be in the drop down under Committees
  • Membership Requests: Choose how you want to handle new members joining. Either "Open" or "Moderated" is recommended as this allows members to self join and the moderator to simply approve (Moderated)
  • Check if you want membership to appear on the site's registration form.
  • Enter Text in the Committee Description. This is for a more detailed description of the group/committee and shows up on the group/committee's main page.
  • Recommend leaving the "List in Groups directory" checked. This makes it easier for members to join. Can always be changed later.
  • Recommend leaving the "Private group" unchecked. This makes it easier for members to join. Can always be changed later
  • Notifications - Leave "Disable Notifications" unchecked. This allows members to determine for themselves if they want to receive email notices of updates and group activity.
  • Theme Configuration - Leave default setting so there will be a consistent site interface
  • Revision information - Leave "Create new revision" checked. This creates a new "copy" of the page. Revision control allows you to go back to previous versions of the content that you are updating.
  • File attachments: Add any files that you would like to link to the main committee page
  • URL Path Settings: Leaving "Automatic alias" checked will have the system create a url path from group/committee title. Uncheck the "Automatic alias" box and enter your own path if you wish
  • Comment Settings: Select if you wish to allow comments on the main committee page. Currently only logged in visitors/members may post comments.
  • Authoring Information: This is where the Group/Committee's Manager or Moderator is set. It defaults to the user that is entering the content but can be set to any user name
  • Publishing options:
    • Select "Published" if the page is ready to "go live" and appear on the site
    • Promoted to front page is irrelevant for this site as groups do not currently appear on the site's front page regardless of this setting.
    • Sticky at top of group home page and other lists will make this group/committee appear at the top of group lists. Recommend leaving this unchecked.
  • Click Save at bottom of page.

Changing a Group's Manager or Moderator

  • Edit the Group Committee
  • Go to Authoring Information: This is where the Group/Committee's Manager or Moderator is set. When adding a group/committee this defaults to the user that is entering the content but can be set to any user name entered in the system. When editing it lists the username of current manager/moderator.
  • Click Save at bottom of page.
