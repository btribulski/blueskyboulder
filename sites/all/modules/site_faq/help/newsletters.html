Newsletters / Email Blasts

A newsletters, similar in form to a story, is a simple method for sending information to a list of site members that belong to a certain group.

Current groups are

  • Realty Owners Association of Stone Harbor Newsletter (Paid and active Members)
  • Stone Harbor Realty Owners Association Non Members(Non Paid and inactive Members)

To add a newsletter

  1. On the Side menu Create Content - Newsletter Issue
  2. Enter the Title of the newsletter. This shows up as the in the newsletter title area (very top portion) of the visitors browsers. This is a key area of Search Engine Optimization so make sure your newsletter titles concisely say what the material is about, but still contain your targeted keywords.  
  3. The main text of the newsletter in "Body" area. (this shows up on the individual image newsletter) 
  4. Input Formats:
    1. Filter HTML is used
      1. Web page addresses and e-mail addresses turn into links automatically.
      2. Allowed HTML tags: <a> <em> <strong> <cite> <code> <ul> <ol> <li> <dl> <dt> <dd>
      3. Lines and paragraphs break automatically
    2. Full HTML (Select this if you use images of any kind of text formatting in your body text)
      1. Allows the use of ALL HTML tags
      2. Web page addresses and e-mail addresses turn into links automatically.
  5. Send Newsletter
    1. TEST FIRST!!!!
    2. Select Send one test newsletter to the test address
    3. Enter the test email address. The user_support@stoneharborroa.com is simply a default and can be replaced with any email address you wish
  6. Email Options
    1. Email format: Select html for best looking emails
    2. Email Priority: Keep None selected
  7. Revision Information: Revision Information allows you to create a new "version" of the node and maintain the old version's information in case you want to go back and restore a previous saved version
    1. Check "Create New Revision" (checked by default) to create a new version and save old version
    2. Enter any notes in the Log Message. Visible only on the edit area and only to those that have the permissions to view.
  8. URL path settings: Sets the URL that will be shown in the visitors browser. Allows for "user and search engine friendly" urls. Search Engine Optimization tip is to try and place keywords that are pertinent to the page's content in the url.
  9. Automatic Alias: If checked will create a url based on the content's title
  10. Comment Settings
  11. File attachments
  12. Authoring Information
  13. Publishing Options settings,
  14. Click Save
