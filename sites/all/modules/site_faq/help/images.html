To add an image

  1. On the Right Hand side menu Create Content - Image
  2. Enter the Title of the Image (this shows up as the one-liner below the image thumbnail in the "gallery" level pages)  
  3. Select the Image Galleries pulldown for the area you want the image to go into.
  4. Clicking inside the "Image" box should pull up the "File Upload" window so you can browse to the picture file on your local computer.
  5. Leave "Rebuild derivative images" unchecked. (this is used during editing when you change the image and want to rebuild the thumbnail and preview files)
  6. Enter any description of the image in "Body" area. (this shows up on the individual image page) 
  7. You can ignore the Input Format, Revision Information, URL path settings, Comment Settings, File attachments, Authoring Information, and Publishing Options settings,
  8. Click Save

To edit an image

From the Gallery level page that contains the image you would like to edit

  1. Click the text "Title" link under the thumbnail image.
  2. Click the Edit tab at the top of the image
  3. Edit the Title as needed
  4. If changing - Select the Image Galleries pulldown for the area you want the image to go into 
  5. If changing the image file itself
    1. Clicking inside the "Image" box should pull up the "File Upload" window so you can browse to the picture file on your local computer.
    2. Check the  "Rebuild derivative images" box. (this will rebuild the thumbnail and preview files off the new image you upload)
  6. Edit any description of the image in "Body" area.
  7. You can ignore the Input Format, Revision Information, URL path settings, Comment Settings, File attachments, Authoring Information, and Publishing Options settings,
  8. Click Save
