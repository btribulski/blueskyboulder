Adding content to the frontpage of the site is fairly easy. Currently the Frontpage is set to show "promoted" News Stories and Events.

What is the difference between a News Story and an Event?

  • A News Story is just a simple article/announcement about whatever you wish.
  • An Event should be used when there is a specific date associated with an announcement. An Event can have the textual detail of a article but will also automatically show up in the web site's Event Calendar.

Directions

  1. From the Left Hand Menu Select - Create Content
  2. Select either News Story or Event
  3. Fill in the Title/Event Name, Date(s), and Body/Description
  4. If you select Committees and have the Public box unchecked this item will only appear on the front page for a logged in user that is a member of that committee. If you leave the default settings everyone can view the information.
  5. In the Publishing options (at the bottom of the page) make sure Published and Promoted to the front page are selected (they are by default)

Helpful Tip: If you ever want to keep and item published on the site but not appear on the frontpage just uncheck Promoted to the front page when editing a News Story or Event
