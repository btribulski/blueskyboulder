Site Pages

A page, similar in form to a story, is a simple method for creating and displaying information that rarely changes, such as an "About us" section of a website. By default, a page entry does not allow visitor comments and is not featured on the site's initial home page.

To add a page

  1. On the Side menu Create Content - Page
  2. Enter the Title of the Page. This shows up as the in the page title area (very top portion) of the visitors browsers. This is a key area of Search Engine Optimization so make sure your page titles concisely say what the material is about, but still contain your targeted keywords.  
  3. The main text of the page in "Body" area. (this shows up on the individual image page) 
  4. Input Formats:
    1. Filter HTML is used
      1. Web page addresses and e-mail addresses turn into links automatically.
      2. Allowed HTML tags: <a> <em> <strong> <cite> <code> <ul> <ol> <li> <dl> <dt> <dd>
      3. Lines and paragraphs break automatically
    2. Full HTML (Select this if you use images of any kind of text formatting in your body text)
      1. Allows the use of ALL HTML tags
      2. Web page addresses and e-mail addresses turn into links automatically.
  5. Menu Settings: User this area if you would like the page to be a link on a menu
    1. Menu Link Title: The text that will show up in the menu
    2. Parent Item: This is where you set where you would like the link to show up. <Primary Links> (or <menu name>) will set the link to be a top level item. If you select another parent item the link will appear on the "mouseover flyout" menu underneath/beside the main link
    3. Weight: Allows you to determine the order in which the link will appear in the menu, the higher numbered items will sink and the lower numbered items will be positioned nearer the top.
  6. Revision Information: Revision Information allows you to create a new "version" of the node and maintain the old version's information in case you want to go back and restore a previous saved version
    1. Check "Create New Revision" (checked by default) to create a new version and save old version
    2. Enter any notes in the Log Message. Visible only on the edit area and only to those that have the permissions to view.
  7. URL path settings: Sets the URL that will be shown in the visitors browser. Allows for "user and search engine friendly" urls. Search Engine Optimization tip is to try and place keywords that are pertinent to the page's content in the url.
  8. Automatic Alias: If checked will create a url based on the content's title
  9. Comment Settings
  10. File attachments
  11. Authoring Information
  12. Publishing Options settings,
  13. Click Save

To edit an Page

  1. Make sure that you are logged in to the site with an account that has access to edit site pages.
  2. Navigate to the page you would like to edit OR click "View/Edit Site's Content" from the side menu
  3. Click the edit tab at the top of the page
  4. Edit the Title as needed
  5. Edit the Body as needed
  6. You can ignore the Input Format, Revision Information, URL path settings, Comment Settings, File attachments, Authoring Information, and Publishing Options settings,
  7. Click Save
